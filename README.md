# 📚 QuickStudy

QuickStudy is an AI-powered study tool built for students and lifelong learners. It helps users generate past questions, personalized summaries, and PDF study guides — all with the help of intelligent prompts and web automation.

> 🧠 Designed & developed by [Imafidon Victory](https://github.com/victory-imafidon) | Built with PHP, JS, MySQL | Deployed as a PWA for mobile use

---

## ✨ Features

- 🧾 **AI-Powered Question Generator** – Custom questions based on any topic
- 📄 **PDF Export** – Study materials downloadable in clean formats
- 🔍 **Smart Search** – Finds and filters relevant results
- 📚 **Past Questions Access** – Nigerian universities & WASSCE/NECO/UTME style
- 🧠 **Prompt-Driven Learning** – LLM-ready backend that adapts over time
- 📱 **Mobile-First PWA** – Usable offline with app-like behavior

---

## 🛠 Built With

- **Backend:** PHP, MySQL
- **Frontend:** Bootstrap, Vanilla JavaScript
- **AI Layer:** Custom prompt engine, future integration with LLM APIs
- **Mobile:** PWA setup for easy mobile installation

---

## 🧠 Motivation

Many students struggle with access to affordable and personalized learning tools. QuickStudy was designed to fill that gap by using AI to turn textbooks and topics into ready-made study guides and practice questions — even offline.
